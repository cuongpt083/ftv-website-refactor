# COMPONENT SPECIFICATION
Project: FINTECHVIET

This document defines reusable UI components used across all templates.

---

# 1. Primary Button

Purpose:
Primary action for the page.

Structure:

button
span label

Variants:

Primary  
Secondary  
Ghost

States:

Default  
Hover  
Focus  
Disabled

Example:

<button class="btn-primary">Talk to Us</button>

---

# 2. Section Header

Purpose:
Introduce a section.

Structure:

eyebrow label  
section title  
optional description

Example:

<div class="section-header">
 <span class="eyebrow">SOLUTIONS</span>
 <h2>Modern Banking Infrastructure</h2>
 <p>Enterprise-grade fintech solutions.</p>
</div>

---

# 3. Hero Section

Purpose:
Primary introduction to the website.

Structure:

hero-container  
headline  
subtext  
primary CTA  
secondary CTA  
hero image

Example markup:

<section class="hero">
 <div class="hero-content">
  <h1>Modern Digital Infrastructure for Finance</h1>
  <p>We build scalable banking and payment technology.</p>
  <button class="btn-primary">Book Consultation</button>
 </div>
</section>

---

# 4. Solution Card

Purpose:
Show core services.

Structure:

icon  
title  
description  
learn more link

<div class="solution-card">
 <h3>Core Banking Modernization</h3>
 <p>Upgrade legacy banking platforms.</p>
 <a href="#">Learn More</a>
</div>

# 8. Logo Strip

Purpose:
Show trusted client logos.

Structure:

horizontal logo list

Mobile behavior:
horizontal scroll

---

# 9. Logo Grid

Purpose:
Show platform and technology capabilities.

Structure:

grid wrapper  
technology logo or label  
optional category label

Display:

2 columns on mobile  
3 to 4 columns on tablet  
6 columns on desktop

---

Purpose:
Encourage contact.

Structure:

headline  
description  
button

Example:

<section class="cta-banner">
 <h2>Ready to modernize your fintech stack?</h2>
 <button>Talk to Us</button>
</section>

---

# 11. Mobile Drawer Menu

Purpose:
Mobile navigation.

Structure:

logo

CTA button

Behavior:

Slide from right

---

# 12. Footer Column

Purpose:
Footer navigation grouping.

Structure:

column title
