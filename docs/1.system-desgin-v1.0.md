---

# 1. Design Principles

## 1.1 Core Philosophy

The website must communicate:

- **Enterprise trust**
- **FinTech expertise**
- **Technical innovation**
- **Conversion-driven UX**
- **Mobile-first usability**

Key design rules:

1. Content clarity over visual decoration
2. Modular section-based architecture
3. Mobile-first layout system
4. Trust signals above the fold
5. CMS-friendly component structure
6. Fast scanning UX for B2B decision makers
